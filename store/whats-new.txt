Satoshi Cloud Miner 1.0.0 — Build #13 (Free Forever plan + Admin AI controls + Logout fix)

What's new in this build:

NEW — Free Forever mining plan
- Every user can now activate a complimentary 500 GH/s mining plan once
  every 24 hours, right from the Home dashboard.
- Tap "Activate Free Plan" to start a 24-hour cycle. While it's running you
  see a live "Plan resets in HH:MM:SS" countdown; once the timer hits zero
  the activation button comes back automatically.

NEW — Operator Console enhancements
- AI Trading Agents are now editable: operators can tap any agent to nudge
  its daily %, win rate, and signal strength, and there's a "Regenerate"
  button that replaces today's snapshot with a fresh randomized one.
- Commission Fees Pool card shows total withdrawal commission collected,
  what's already been reinvested, and the unreinvested balance. A single
  tap on "Reinvest" credits the available pool to the operator's balance
  and logs the action in the audit trail.

FIXED — Sign-out routing
- Sign-out now navigates to the welcome screen immediately and reliably.
  Build #12's deferred-redirect could occasionally leave the screen black
  on certain iOS versions; that path has been replaced with a direct
  router.replace from inside the auth context so the auth screen mounts
  before any state teardown.

Previous changes still in place:
- Real Apple In-App Purchase via StoreKit 2 for all 10 mining plans and
  the Ad-Free $3.99 tier.
- Lightning withdrawals (BOLT11 + Lightning Address) with 150,000 sats
  minimum, 10% flat fee.
- Server-side daily check-in cron and optional auto-reinvest.
- AI Trading Agents simulation with daily P&L + win-rate per strategy.
- AI market commentary ticker refreshed daily.
